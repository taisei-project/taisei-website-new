We provide pre-built packages for Windows, macOS, and some Linux distributions.

You can also download the source code and build it yourself, or look for a package in your distribution's repositories.

If you're looking for PGP signatures, SHA256 sums, or older releases, visit the releases page on GitHub.

If you find any mistakes, please contact us.

Source Code

Building Taisei is easy (unless you are on Windows).

Download the source code and follow the instructions in README.md.

Feel free to contact us if you have problems.

Although Taisei supports only Linux, macOS, and Windows officially, it's also known to work on BSD systems and Haiku.

Generally, it should build and run on any POSIX-compliant OS, given that all the dependencies also work there.
